# 📐 TIÊU CHUẨN THIẾT KẾ XÂY DỰNG VIỆT NAM
## Dành cho nhà ở dân dụng — Cập nhật từ TCVN + QCVN

---

## 1. KÍCH THƯỚC TỐI THIỂU (TCVN 4451:2012)

| Hạng mục | Tối thiểu | Ghi chú |
|----------|:---------:|---------|
| 🛋️ Phòng khách | 14m² | — |
| 🛏️ Phòng ngủ | 9-12m² | Tối thiểu 9m², nên 12m² |
| 🍳 Bếp + ăn | 10m² | — |
| 🚻 WC | 2.5m² | Rộng tối thiểu 1.2-1.5m |
| 🚿 Phòng tắm | 2.5-3m² | Có thể gộp WC |
| 🪜 Cầu thang | Rộng ≥ 0.9m | Bậc cao 150-180mm, rộng ≥250mm |
| 🚪 Cửa chính | 0.9m x 2.0m | Rộng x cao |
| 🪟 Cửa sổ | ≥ 10% diện tích sàn | Chiếu sáng tự nhiên |
| 📏 Chiều cao trệt | ≥ 3.0m | Thông thủy |
| 📏 Chiều cao lầu | ≥ 2.7m | Thông thủy |

## 2. QUY ĐỊNH XÂY DỰNG CƠ BẢN

| Hạng mục | Quy định |
|----------|----------|
| Mật độ xây dựng | Theo quy hoạch từng khu vực |
| Chỉ giới đường đỏ | Không xây quá lộ giới |
| Khoảng lùi | Tùy theo đường rộng (QCVN 01:2021) |
| Chiều cao tối đa | Theo quy hoạch đô thị |
| PCCC | QCVN 06:2022/BXD |
| Thông gió | TCVN 5687:2010 |

## 3. BẢN VẼ THIẾT KẾ

| Loại bản vẽ | Nội dung |
|-------------|----------|
| Mặt bằng (Floor Plan) | Bố trí phòng, cửa, cầu thang, kích thước |
| Mặt đứng (Elevation) | Hình chiếu đứng — nhìn từ ngoài |
| Mặt cắt (Section) | Cắt dọc/ngang công trình |
| Phối cảnh 3D | Hình ảnh 3 chiều |
| Bản vẽ kết cấu | Móng, cột, dầm, sàn |
| Bản vẽ điện | Sơ đồ điện, ổ cắm, đèn |
| Bản vẽ nước | Cấp thoát nước |
| Bản vẽ chi tiết | Cửa, cầu thang, WC |

## 4. NHÀ KHUNG THÉP

| Hạng mục | Thông số |
|----------|----------|
| Khoảng cách cột | 4-6m (tùy tải) |
| Dầm thép | I100-I300 (tùy nhịp) |
| Sàn | Đổ bê tông + tôn sàn |
| Mái | Tôn lạnh / Tôn cách nhiệt |
| Tường | Gạch nhẹ / Panel |
| Chống thấm | Bắt buộc mái + WC |

## 5. LƯU Ý THIẾT KẾ CHO VLHT

- WC cần ≥ 2.5m², rộng tối thiểu 1.2-1.5m
- Cầu thang cần rộng ≥ 0.9m
- Chiều cao tầng nên 3.0-3.6m (không cần 4m)
- Phòng ngủ tối thiểu 12m²
- Cửa chính showroom nên rộng 1.2-1.8m
- Cần bố trí cửa sổ cho tất cả phòng (≥10% diện tích)
