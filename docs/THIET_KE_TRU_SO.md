# 🏢 THIẾT KẾ TRỤ SỞ CÔNG TY VLHT TRẦN HỮU MINH

## 📐 Thông số tổng quan
| Hạng mục | Chi tiết |
|----------|----------|
| **Loại công trình** | Nhà khung thép 3 tầng |
| **Diện tích đất** | 6m x 16m = **96m²** |
| **Tổng diện tích sàn** | ~ **120m²** (tầng 1 thụt 1m) |

---

## 🏗️ TẦNG 1 — Showroom + Để xe (5m x 16m = 80m²)
| Khu vực | Kích thước | Diện tích |
|---------|------------|:---------:|
| 🚗 Để xe (thụt 1m) | 6m x 1m | 6m² |
| 🛋️ Showroom cửa hàng | 5.5m x 11.5m | ~63m² |
| 🚻 WC + Nhà tắm | 0.5m x 11.5m | ~5.7m² |
| 🪜 Cầu thang lên tầng | 6m x 3.5m | 21m² |

**Đặc điểm:**
- Thụt vào 1m phía trước làm chỗ đậu xe + mái che
- Showroom rộng trưng bày sơn, vật liệu
- WC + nhà tắm kết hợp dạng dài (0.5m ngang) cuối nhà
- Cầu thang lên tầng 2 ở cuối

---

## 🏗️ TẦNG 2 — Showroom (6m x 16m = 96m²)
| Khu vực | Kích thước | Diện tích |
|---------|------------|:---------:|
| 🛋️ Showroom tầng 2 | 5.5m x 12m | ~66m² |
| 🚻 WC | 0.5m x 2.5m | ~1.2m² |
| 🪜 Cầu thang lên tầng 3 | 6m x 4m | 24m² |

**Đặc điểm:**
- Showroom tầng 2 trưng bày thêm sản phẩm
- WC nhỏ ở cuối
- Cầu thang lên tầng 3

---

## 🏗️ TẦNG 3 — Phòng ngủ (6m x 16m = 96m²)
| Khu vực | Kích thước | Diện tích |
|---------|------------|:---------:|
| 🛏️ Phòng ngủ 1 | 5.5m x 5m | ~27.5m² |
| 🛏️ Phòng ngủ 2 | 5.5m x 5m | ~27.5m² |
| 🚻 WC + Nhà tắm | 0.5m x 5.5m | ~2.7m² |
| 🪜 Cầu thang | 2m x 6m | 12m² |
| 🛋️ Sảnh + Hành lang | 4m x 6m | 24m² |

**Đặc điểm:**
- 2 phòng ngủ rộng rãi
- WC + nhà tắm chung
- Sảnh + hành lang rộng

---

## 📋 Thông tin thêm
- 📍 **Mặt tiền đường** (hướng 6m)
- 🚗 **Có chỗ đậu xe** (thụt 1m phía trước)
- 🪟 **Cửa kính** mặt tiền showroom
- 🔧 **Khung thép** chịu lực, có thể mở rộng sau này
